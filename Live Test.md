## 🔑 How to get a FREE Google Gemini API Key

To use the application, you will need a Gemini API key. It's completely free and takes just a minute to set up!

**Step-by-step Guide:**
1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Sign in with your Google account.
3. On the left sidebar, click on **"Get API key"**.
4. Click the blue **"Create API key"** button.
5. Copy the generated key.
6. Open the [Live App](https://ragpdf-apj2bpm3tjrnwsrkm7ppkc.streamlit.app/) and paste your key into the sidebar.
7. Upload a PDF and start asking questions!
